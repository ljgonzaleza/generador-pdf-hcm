# Carpeta de código fuente (abapGit)

Aquí irán los objetos ABAP en formato abapGit SOLO cuando se inicie la fase de
codificación (palabra clave CODIFICAR). Ejemplos de nombres de archivo:

  zhhrr_775.prog.abap          (programa)
  zhhrr_775.prog.xml           (metadatos del programa)
  zhcm_cp_tplt.tabl.xml        (tabla)
  zcl_cp_pdf_generator.clas.abap   (clase)

Un objeto = un archivo. No crear nada aquí hasta la fase de código.
